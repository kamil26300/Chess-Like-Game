
## Installation

Install all server side dependencies with npm

```bash
  cd server
  npm install
```
Run backend by executing
```bash
  npm start
```

Install all client side dependencies with npm

```bash
  cd client
  npm install
```
Start react project by entering
```bash
  npm start
```
